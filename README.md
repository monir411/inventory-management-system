# Inventory & Sales ERP System

A high-performance, premium ERP system for managing inventory, sales operations, and company-wise product distribution.

---

## 🚀 Key Features

### 🏢 Company & Product Management
*   **Company Profiles:** Manage multiple companies with specific configurations.
*   **Product Catalog:** Linked to companies with SKU tracking, buy/sell pricing, and unit management.

### 📦 Advanced Stock Workspace
*   **Real-time Tracking:** Live inventory balances calculated from all stock movements.
*   **Intelligent Alerts:** Automated tracking for Low Stock and Zero Stock items.
*   **Movement History:** Detailed audit logs of all entries (Opening Stock, Stock In, Adjustments, Sales, Returns).
*   **Loss Prevention:** Dedicated module for recording and analyzing stock damages.

### 💰 Sales & Collection System
*   **Sale Creation:** Multi-item sales with automated profit calculation and invoice generation.
*   **Payment Handling:** Support for partial payments and real-time due tracking.
*   **Distribution Tracking:** Route-wise and Shop-wise performance and due summaries.
*   **Printable Reports:** Professional Daily Sales Summary reports (A4 optimized) with company-wise filtering.

### 📊 Executive Dashboard
*   **Core Metrics:** High-visibility KPI cards for Today's Sales, Profit, and Collections.
*   **Asset Valuation:** Real-time calculation of total capital investment in stock.
*   **Efficiency Tracking:** Collection efficiency percentage and loss prevention metrics.
*   **Performance Progress:** Visual revenue and stock distribution bars across all companies.

---

## 🛠️ Technical Excellence

### Performance Optimization
*   **TanStack Query (React Query):** Implemented for professional server-state management. Provides instant caching, background synchronization, and automatic deduplication.
*   **Global Sync Indicator:** A premium glassmorphism loader that tracks background data operations without blocking user interaction.
*   **Debounced Search:** Optimized search inputs to minimize API churn and ensure a snappier UI feel.
*   **Segmented Loading:** Section-level skeleton loaders to improve perceived performance.

### Tech Stack
*   **Frontend:** Next.js, Tailwind CSS, TanStack Query, Lucide Icons.
*   **Backend:** Node.js (NestJS architecture), REST API.
*   **Security:** Token-based Authentication system with Role-Based Access Control (RBAC).
